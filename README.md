# RAG Document Search AI

A simple and efficient Retrieval-Augmented Generation (RAG) application. This project allows users to upload documents (PDF, DOCX, TXT) and ask questions. The AI answers strictly based on the uploaded content using Google Gemini and LangChain.

## Features

Multi-File Upload: Supports PDF, Word, and Text files.
Smart Retrieval: Uses FAISS to find the most relevant parts of your documents.
Context-Aware: The AI is programmed to only answer from your files to ensure accuracy.
Easy UI: A clean chat interface built with Streamlit. 

## Tech Stack

Language: Python,
Framework: LangChain,
Model: Google Gemini (Generative AI),
Vector DB: FAISS,
Interface: Streamlit
